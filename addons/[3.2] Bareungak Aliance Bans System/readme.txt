* 안내사항 *

에드온 폴더속 lua\autorun\server\babs_sv_init.lua 파일 속 SteamAPI 을(를) 직접 입력해주셔야 부계정 차단기능을 사용하실 수 있습니다.
	https://steamcommunity.com/dev/apikey <---- 이곳에서 발급받아 큰따옴표 사이에 복사해주세요. 

인터페이스 명령어
    !연합밴 : 연합밴 인터페이스를 실행 시킵니다.

기본 명령어 (모두 콘솔 "~" 키를 눌러서 입력해주세요)
    babs_interface : 연합밴 인터페이스를 실행 시킵니다.
    babs_getversion : 에드온의 현재 최신버전과 서버에 설치된 에드온의 버전을 보여줍니다.
    babs_update : 수동으로 데이터베이스를 업데이트 시킵니다.

연합밴 대상자가 차단된 후 콘솔에 표출되는 오류는 정상입니다.
